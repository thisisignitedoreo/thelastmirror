Attack On Titan Tribute Game

Attack On Titan Tribute Game (Вторжение титанов) – игровой проект, который был создан по мотивам одноименного популярного во всем мире аниме. Здесь тебе предстоит отправится на спасение мира, а для этого необходимо уничтожить всех вражеских титанов до единого. Эти титаны представляют из себя неких существ, которые явно похожи на людей я но обладают огромными размерами тела. Тебя ждёт огромный открытый мир, передвигаться по которому ты сможешь с помощью специальных тросов, которые будут установлены на спине твоего главного героя. Для того, чтобы с лёгкостью одолеть воздушное пространство, тебе всего-то стоит нажимать клавиши Q и Е, которые предназначены для левого и правого троса. Чтобы одновременно выпустить сразу два троса, тебе необходимо воспользоваться клавишей под названием пробел. Для передвижения по той или ной локации, используй различные строения, которые будут встречаться на твоём пути.
